new Firebase( )

Construct a new Firebase reference from a full Firebase URL. See Creating Firebase References.

Arguments

new Firebase( firebaseURL )
firebaseURL : The absolute, HTTPS URL of a Firebase location (e.g. https://SampleChat.firebaseIO-demo.com/users).

Returns

A Firebase reference for the specified location.

Usage

var usersRef = new Firebase('https://SampleChat.firebaseIO-demo.com/users');
// usersRef now refers to the 'users' Firebase location.
