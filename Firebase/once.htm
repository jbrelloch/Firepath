once( )

Listens for exactly one event of the specified event type, and then stops listening. This is equivalent to calling “on”, and then calling “off” inside the callback function.

See on( ) for details on event types.

Arguments

once( eventType, successCallback, [failureCallback], [context] )
eventType : One of the following strings: 'value', 'child_added', 'child_changed', 'child_removed' or 'child_moved'. successCallback : The callback. failureCallback (optional) : If provided, the failureCallback will fire (instead of the successCallback) if you don't have permission to access this data. context (optional) : If provided, this object will be used as this when calling your callback.

Returns

This function does not return anything.

Usage

// Basic usage of .once() to read the data located at firebaseRef.
firebaseRef.once('value', function(dataSnapshot) { /* handle read data. */ });

// Providing 'this' as the context argument in order to call a local method.
firebaseRef.once('value', this.onReadData, this);
